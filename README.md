# theweather
Command line tool for displaying weather at the provided address

[Powered by Dark Sky](https://darksky.net/poweredby)

### Usage Examples:
$ ./weather.py 10108

$ ./weather.py "London, UK"

$ ./weather.py "123 Sesame St"

### Output Example:
Current conditions for Los Angeles, CA:

Summary:     Clear

Temperature: 87.41
